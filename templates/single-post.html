Detectron2

Detectron2 is Facebook AI Research's next generation software system that implements state-of-the-art object detection algorithms. It is a ground-up rewrite of the previous version, Detectron, and it originates from maskrcnn-benchmark.

Multiple models are available for implementation. Again, due to technical limitations, this project selects the lowest memory-consuming one compatible with COCO object detection: faster_rcnn_R_50_FPN_1x

With limited data available to the model (local memory limitations) and the risk of OOM errors from long iterations, the results are not expected to be particularly high.
